Promises

In the Javascript I have a promise that runs after 5 seconds and prints a message to the console. 

    
    let sayHello = new Promise(function (resolve, reject) {
      setTimeout(() => {
        resolve("Hello world");
      }, 5000)
    });

    sayHello.
      then((msg) => {
        console.log(msg)
        return msg
      }).
      then((msg) => {
        msg += " and hello universe"
        console.log(msg);
      })
